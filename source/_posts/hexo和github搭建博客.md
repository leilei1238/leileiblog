### hexo环境搭建
- 安装npm install -g hexo-cli
- hexo init
- npm install 
- hexo server 启动服务
### hexo常用命令
- hexo init [folder]：新建一个网站，如果没有设置folder,默认在当前文件夹下建立
- hexo new 文章名字
- hexo new page about:创建"关于我们"页面
- hexo generate 生成静态文件：`-d,--deploy`文件生成后立即部署网站；`-w,--watch`监视文件变动
